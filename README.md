<h1>How to use.</h1>

<h3>open code space</h3>
  - fisrt click the button at the top of the repo that says code
  
  - then click the tab which says "codespace"
    
  - then click "create new codespace"


<h3>how to open repos in the codespace</h3>

  - first navigate to a new terminal (`Ctrl+Shift+'`)
    
  - then run the script but typing `./github-login.sh`
    
  - then follow instructions.
